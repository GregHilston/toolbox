- split vertically $mod + v before $mod + enter
- split horizontally $mod + h before $mod + enter
- toggle fullscreen mode $mod + f
- open dmenu (handy way to open applications) $mod + d
- close window $mod + q
- kill window $mod + shift + q

    - web browser
